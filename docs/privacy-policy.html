SnoreLess 개인정보처리방침

시행일: 2026년 3월 6일 | 최종 수정: 2026년 3월 6일

SnoreLess는 사용자의 모든 데이터를 기기 안에서만 처리합니다. 외부 서버로 전송되는 정보는 없습니다.

1. 수집하는 정보

정보 유형 수집 목적 저장 방식
마이크 오디오 코골이 소리 실시간 감지(dB 측정) 실시간 처리 후 즉시 폐기. 기기에 저장하지 않음
코골이 이벤트 데이터 코골이 횟수, 시간, 강도 기록 기기 내 로컬 저장(SwiftData)
코골이 구간 녹음 사용자 본인 확인용 짧은 녹음 기기 내 로컬 저장. 서버 전송 없음
건강 데이터(HealthKit) 수면 분석 연동 Apple HealthKit 프레임워크 내 저장(로컬)

2. 외부 서버 전송

SnoreLess는 수집한 어떠한 데이터도 외부 서버로 전송하지 않습니다. 모든 데이터 처리는 100% 사용자의 기기(iPhone, Apple Watch) 안에서 이루어집니다.

3. 제3자 제공

사용자의 개인정보를 제3자에게 제공하지 않습니다. 광고 SDK, 분석 SDK 등 외부 서비스를 사용하지 않습니다.

4. 파트너 공유 기능

앱 내 "개선 현황 공유" 기능은 사용자가 직접 선택한 상대에게만 요약 데이터를 공유합니다. 이 과정에서도 데이터는 Apple의 기기 간 통신(iMessage, AirDrop 등)을 통해 전달되며, SnoreLess 서버를 거치지 않습니다.

5. 데이터 보관 및 삭제

  • 모든 데이터는 기기 내 SwiftData에 저장됩니다.
  • 앱을 삭제하면 모든 데이터가 함께 삭제됩니다.
  • 앱 설정에서 언제든 녹음 데이터 및 이벤트 기록을 개별 삭제할 수 있습니다.

6. 마이크 및 백그라운드 오디오 권한

SnoreLess는 다음 목적으로 마이크 접근 권한과 백그라운드 오디오 권한을 요청합니다:

  • 마이크: 수면 중 코골이 소리의 데시벨(dB)을 측정하여 코골이 여부를 판단합니다.
  • 백그라운드 오디오: 수면 중에도 지속적으로 코골이를 모니터링하기 위해 백그라운드에서 오디오 세션을 유지합니다.

마이크로 수집되는 오디오 스트림은 dB 분석 후 즉시 폐기되며, 사용자가 명시적으로 "녹음 저장"을 활성화한 구간만 로컬에 저장됩니다.

7. 아동 개인정보

SnoreLess는 만 14세 미만 아동의 개인정보를 수집하지 않습니다.

8. 개인정보처리방침 변경

본 방침이 변경될 경우, 앱 내 공지 및 이 페이지를 통해 사전 안내합니다.
